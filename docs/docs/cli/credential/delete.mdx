# pf credential delete

## Usage

```bash
pf credential delete [OPTIONS] CREDENTIAL_ID
```

## Summary

Deletes a credential.

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`credential_id`** | UUID | ID of credential to delete. | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--force`, `-f` | BOOLEAN | Forcefully deletes credential without confirmation prompt. | False | ❌ |
