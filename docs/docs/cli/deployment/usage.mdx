# pf deployment usage

## Usage

```bash
pf deployment usage YEAR MONTH DAY
```

## Summary

Shows the usage of all deployments in a project within a specific month or day.

:::info
- Timestamps are recorded in UTC.
- The usages are updated every minute.
:::

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`year`** | INTEGER | None | - | ✅ |
| **`month`** | INTEGER | None | - | ✅ |
| `day` | INTEGER | None | None | ❌ |
