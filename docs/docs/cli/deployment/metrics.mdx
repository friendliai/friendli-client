# pf deployment metrics

## Usage

```bash
pf deployment metrics [OPTIONS] DEPLOYMENT_ID
```

## Summary

Show metrics of a deployment.

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`deployment_id`** | TEXT | ID of deployment to inspect in detail. | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--time-window`, `-t` | INTEGER | Time window of metrics in seconds. | 60 | ❌ |
