# pf deployment metrics

## Usage

```bash
pf deployment metrics [OPTIONS] DEPLOYMENT_ID
```

## Summary

Show metrics of a deployment.

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`deployment_id`** | TEXT | ID of deployment to inspect in detail. | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| **`--since`** | TEXT | Start time of metrics to fetch. The format should be {YYYY}-{MM}-{DD}T{HH}. The UTC timezone will be used by default. | - | ✅ |
| **`--until`** | TEXT | End time of metrics to fetch. The format should be {YYYY}-{MM}-{DD}T{HH}. The UTC timezone will be used by default. | - | ✅ |
| `--window`, `-w` | INTEGER | Time window of metrics in seconds. | 60 | ❌ |
