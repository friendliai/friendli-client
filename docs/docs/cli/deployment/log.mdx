# pf deployment log

## Usage

```bash
pf deployment log DEPLOYMENT_ID REPLICA_INDEX
```

## Summary

Shows deployments log.

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`deployment_id`** | TEXT | ID of deployment to get log. | - | ✅ |
| `replica_index` | INTEGER | Replica index of deployment to get log. | 0 | ❌ |
