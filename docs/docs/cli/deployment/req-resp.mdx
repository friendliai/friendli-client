# pf deployment req-resp

## Usage

```bash
pf deployment req-resp [OPTIONS] DEPLOYMENT_ID
```

## Summary

Downloads request-response logs for a deployment.

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`deployment_id`** | TEXT | ID of deployment to download request-response logs. | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| **`--since`** | TEXT | Start time of request-response logs. The format should be {YYYY}-{MM}-{DD}T{HH}. The UTC timezone will be used by default. | - | ✅ |
| **`--until`** | TEXT | End time of request-response logs. The format should be {YYYY}-{MM}-{DD}T{HH}. The UTC timezone will be used by default. | - | ✅ |
| `-s`, `--save-dir` | TEXT | Directory path to save request-response logs | None | ❌ |
