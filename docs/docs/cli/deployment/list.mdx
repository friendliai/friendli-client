# pf deployment list

## Usage

```bash
pf deployment list [OPTIONS]
```

## Summary

Lists all deployments.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--include-terminated` | BOOLEAN | When True, shows all deployments in my project including terminated deployments. (active deployments are shown above the terminated ones.) | False | ❌ |
| `--limit` | INTEGER | The number of deployments to view | 20 | ❌ |
| `--from-oldest` | BOOLEAN | When True, shows from the oldest deployments. | False | ❌ |
| `--org` | BOOLEAN | Shows all deployments in org | False | ❌ |
