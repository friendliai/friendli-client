# pf passwd

## Usage

```bash
pf passwd [OPTIONS]
```

## Summary

Change password.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| **`--old-password`** | TEXT | None | - | ✅ |
| **`--new-password`** | TEXT | None | - | ✅ |
| **`--confirm-password`** | TEXT | None | - | ✅ |
