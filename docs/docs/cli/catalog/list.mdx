# pf catalog list

## Usage

```bash
pf catalog list [OPTIONS]
```

## Summary

Lists public checkpoints in catalog.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--name`, `-n` | TEXT | The name of public checkpoint to search. | None | ❌ |
| `--limit`, `-l` | INTEGER | The number of public checkpoints to display. | 20 | ❌ |
