# pf org accept-invite

## Usage

```bash
pf org accept-invite [OPTIONS]
```

## Summary

Accept organization invitation.

When the invitee accepts the invitation, the invitee becomes a **Member** of the invited
organization.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| **`--token`** | TEXT | None | - | ✅ |
| **`--key`** | TEXT | None | - | ✅ |
