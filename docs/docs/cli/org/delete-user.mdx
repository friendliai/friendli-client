# pf org delete-user

## Usage

```bash
pf org delete-user [OPTIONS] EMAIL
```

## Summary

Remove a user from the organization.

:::info
Only the **Owner** of the organization can remove a member.
:::

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`email`** | TEXT | Email of the user to delete from the organization | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--force`, `-f` | BOOLEAN | Forcefully delete without confirmation prompt | False | ❌ |
