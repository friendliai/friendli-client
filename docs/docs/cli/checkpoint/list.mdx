# pf checkpoint list

## Usage

```bash
pf checkpoint list [OPTIONS]
```

## Summary

Lists all checkpoints that belong to a user's organization.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--source`, `-s` | CHOICE: [USER, JOB, COPY, REF] | Source of checkpoints. | None | ❌ |
| `--limit`, `-l` | INTEGER | The number of recent checkpoints to see. | 20 | ❌ |
| `--deleted`, `-d` | BOOLEAN | Shows deleted checkpoint. | False | ❌ |
