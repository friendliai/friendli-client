# pf checkpoint download

## Usage

```bash
pf checkpoint download [OPTIONS] CHECKPOINT_ID
```

## Summary

Downloads checkpoint files to local storage.

:::info
You cannot download checkpoints linked from your cloud storage.
:::

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`checkpoint_id`** | UUID | ID of checkpoint to download. | - | ✅ |

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--output`, `-o` | TEXT | Output path to the directory to save checkpoint files. | None | ❌ |
