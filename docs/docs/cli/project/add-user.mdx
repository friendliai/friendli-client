# pf project add-user

## Usage

```bash
pf project add-user EMAIL ROLE
```

## Summary

Add user to project.

If you are the **Admin** of a project, you can add members in the same organization
to the project and assign their roles in the project.

:::info
If you have **Owner** role in the organization, you have full-access to all projects
inside the organization.
:::

## Arguments

| Argument | Type | Summary | Default | Required | 
|----------|------|---------|---------|----------|
| **`email`** | TEXT | Email of the user to add to the current working project | - | ✅ |
| **`role`** | CHOICE: [admin, maintain, develop, guest] | Project role to assign | - | ✅ |
