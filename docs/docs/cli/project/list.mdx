# pf project list

## Usage

```bash
pf project list [OPTIONS]
```

## Summary

List all accessible projects.

## Options

| Option | Type | Summary | Default | Required |
|--------|------|---------|---------|----------|
| `--tail` | INTEGER | The number of project list to view at the tail | None | ❌ |
| `--tail` | INTEGER | The number of project list to view at the head | None | ❌ |
| `--group`, `-g` | BOOLEAN | Show all projects in the current group | False | ❌ |
