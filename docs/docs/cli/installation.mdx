# ðŸ’¾ Installation

You can simply install `periflow-client` package using `pip`.

```bash
pip install periflow-client
```

:::info
`periflow-client` requires **python>=3.8**.
:::

:::info
We recommend using the most up-to-date package.
You can check the release history at [PyPI](https://pypi.org/project/periflow-client/#history) and [Github](https://github.com/friendliai/periflow-client/releases).
You can update the package with:

```bash
pip install periflow-client -U --no-cache-dir
```
:::
