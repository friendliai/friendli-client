You can install PeriFlow Client Python SDK from PyPI.

```bash
pip install periflow-client
```

You can also install from source by cloning the Github repository:

```bash
git clone https://github.com/friendliai/periflow-client.git
cd periflow-client
poetry install
```

:::info
We recommend keeping the package up to date. You can check the release history at [PyPI](https://pypi.org/project/periflow-client/#history) and [Github](https://github.com/friendliai/periflow-client/releases).
You can update the package with:

```bash
pip install periflow-client -U
```
:::
